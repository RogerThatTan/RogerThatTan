## Professional Snapshot

I am a **Software Quality Assurance Trainee** from Dhaka, Bangladesh, focused on building strong expertise in **manual testing, automation testing, API testing, test documentation, defect tracking, and quality-focused engineering**.

My current goal is to grow into a strong **SQA Automation Engineer** by combining practical QA experience with automation, API testing, performance testing basics, and modern software engineering practices.

I am also interested in **AI, deep learning, computer vision, and research**, with a long-term vision to connect software quality engineering with intelligent testing systems.
